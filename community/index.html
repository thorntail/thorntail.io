Community

Thorntail is open source software with a vibrant community. Our community values contributions from all its members, as such we have a Code of Conduct.

If you are interested in contributing to Thorntail, that’s great! We love contributions! To help make it easier getting started, we have a contribution guide on how to setup your environment and submit changes for review. You can find some ways to get started with contributing by picking an issue marked getting-started.

If you are getting started and have questions, would like to contribute to the development of the project, or just want to chat with other users and developers, join us on our Google Group.
